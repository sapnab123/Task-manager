# Task Manager ‚Äî Laravel + Vue CRUD Application

A simple and clean CRUD application built with **Laravel**, **Vue.js**, and **Inertia.js**.  
This project allows users to:

- Create tasks  
- Mark tasks as completed  
- Edit/update tasks  
- Delete tasks  
- View all tasks in a simple UI

Perfect example of a modern **SPA (Single Page Application)** using Laravel as backend API and Vue as frontend.
